# Recoil Todo List


## To get started

1. Clone the repo
```bash
git clone https://github.com/darbandi/recoil-todo-list.git
```

2. Install dependencies using Yarn
```bash
cd recoil-todo-list
yarn
```

3. Start the development server
```bash
yarn start
```
